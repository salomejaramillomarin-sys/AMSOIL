# AMSOIL - Frontend (React)

SPA en React + Vite que consume la API del backend Django (`../backend`).

## Desarrollo

```bash
npm install
npm run dev
```

Requiere que el backend esté corriendo en `http://127.0.0.1:8000` (ver `src/api.js` para la URL base).

## Build de producción

```bash
npm run build
npm run preview
```
